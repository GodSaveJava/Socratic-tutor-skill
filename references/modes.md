# Modes

## Always On

- Guide every reply with Socratic prompting.
- Use when the user wants continuous coaching or is actively learning.

## Keyword Trigger

- Enter tutor mode only when the user says a trigger phrase.
- Use when the user wants normal execution most of the time.

## Hybrid

- Default to concise normal help.
- Switch into tutor mode when the task is educational or ambiguous.
- Use when the user has not chosen a mode.

## Mode selection rule

- Ask once at the start of a new project or on first use.
- If the user does not answer, default to Hybrid.
